# ParabolaWhat?

**ParabolaWhat?** is an interactive educational web application designed to help users master the concepts of quadratic functions and parabolas. Through various exercise modes, users can test and improve their understanding of the relationship between parabolic graphs and their equations.

## Features

-   **Interactive Drawing**: Draw parabolas directly on the screen and get instant feedback on your accuracy (concavity, vertex, intercepts).
-   **Multiple Quiz Modes**:
    -   **Draw the Plot**: Sketch the graph for a given equation.
    -   **Pick the Plot**: Choose the correct graph from multiple options.
    -   **Pick the Equation**: Select the matching equation for a displayed graph.
    -   **Write the Equation**: Analyze a graph and type its full algebraic equation.
    -   **Random Mode**: A mix of all challenges to test overall knowledge.
-   **Gamification**: Earn points, build streaks, and track your accuracy over time.
-   **Responsive Design**: optimized for both desktop and mobile devices.
